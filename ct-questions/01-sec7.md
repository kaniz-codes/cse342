## Bank Account Management System (Java)

Write a Java program to implement a **Bank Account Management System** where a class named **`BankAccount`** manages customer accounts.

The class must use **encapsulation** by keeping account details such as **account number**, **account holder name**, and **balance** as **private variables** with appropriate **getter and setter methods**.
A **constructor** should initialize these values when an account object is created, using the **`this`** keyword to differentiate between instance variables and constructor parameters.
Include a **static variable** to keep track of the **total number of accounts created**, and update it inside the constructor.
Provide methods for **depositing** and **withdrawing** money while ensuring the balance does not become negative, and include a **static method** to display the total number of accounts.

In the `main()` method:
- Create multiple account objects  
- Perform transactions (deposit and withdraw)  
- Display account details  
- Show the total number of accounts created  

Solution:

```java
class BankAccount {
    private String accountNumber;
    private String holderName;
    private double balance;

    private static int totalAccounts = 0;

    public BankAccount(String accountNumber, String holderName, double balance) {
        this.accountNumber = accountNumber;
        this.holderName = holderName;
        if (balance < 0) {
            this.balance = 0;
        } else {
            this.balance = balance;
        }
        totalAccounts++;
    }

    public String getAccountNumber() {
        return accountNumber;
    }

    public void setAccountNumber(String accountNumber) {
        this.accountNumber = accountNumber;
    }

    public String getHolderName() {
        return holderName;
    }

    public void setHolderName(String holderName) {
        this.holderName = holderName;
    }

    public double getBalance() {
        return balance;
    }

    public void setBalance(double balance) {
        if (balance >= 0) {
            this.balance = balance;
        }
    }

    public void deposit(double amount) {
        if (amount > 0) {
            balance += amount;
        }
    }

    public boolean withdraw(double amount) {
        if (amount > 0 && amount <= balance) {
            balance -= amount;
            return true;
        }
        return false;
    }

    public void displayAccountDetails() {
        System.out.println("Account Number: " + accountNumber);
        System.out.println("Holder Name: " + holderName);
        System.out.println("Balance: " + balance);
        System.out.println();
    }

    public static void displayTotalAccounts() {
        System.out.println("Total Accounts Created: " + totalAccounts);
    }
}

public class task3 {
    public static void main(String[] args) {
        BankAccount a1 = new BankAccount("154", "Kaniz", 5000);
        BankAccount a2 = new BankAccount("111", "Fatema", 2000);

        a1.deposit(1500);
        a1.withdraw(1000);

        a2.deposit(500);
        boolean ok = a2.withdraw(4000);
        if (!ok) {
            System.out.println("Withdraw failed: insufficient balance.\n");
        }

        a1.displayAccountDetails();
        a2.displayAccountDetails();

        BankAccount.displayTotalAccounts();
    }
}
```
